## 选择器的进阶用法
### 后代选择器
定义：根据HTML的嵌套关系，选择父元素后代中满足条件的元素。

父亲选择器 \<space\> 儿子选择器

```css
div p{
    color: red;
}
```
Note: 父亲选择器里面所以满足条件的子代都会相应。

### 子代选择器
选择器1 > 选择器2 {css}

```css
div>a{
    color: red;
}
```
解释：只把div里面的儿子选择器的颜色属性改变成红色。

### 并集选择器
多个选择器并列被选择，用逗号分隔。
```css
p, 
div, 
span {
    color: red; 
}
```

### 交集选择器
多个选择器连着写起来。

### hover选择器
解释：选中鼠标悬停在元素上的状态，设置样式
