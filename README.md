# Git Reminder

```bash
git init 
git status -s

git add file
git add . 

git commit -m "messaage"

git rm -f file
git rm --cached file

git log --pretty=oneline
git reset --hard <id>
git reflog --pretty=oneline

git remote add origin https://github.com/guanyuelee/web-code.git

git branch -M main

git push -u origin main

```
